Приёмник данных сайта: вывод контактных данных и отчёт статуса подключения.

== Description ==

Плагин выводит контактные данные сайта и периодически (~1 раз в минуту) отправляет
подписанный отчёт о статусе подключения на настроенный адрес приёма.

Вкладка «Подключение»: адрес приёма, идентификатор сайта, секретный ключ (показывается
один раз при выдаче), проверка соединения.

Надёжность на низкотрафичных сайтах: WP-Cron срабатывает только при посещениях. Для
стабильной отправки раз в минуту отключите псевдо-крон и вызывайте wp-cron.php системным cron:

  define('DISABLE_WP_CRON', true);   // в wp-config.php

  # crontab (каждую минуту):
  * * * * * wget -q -O - https://ВАШ-САЙТ/wp-cron.php?doing_wp_cron >/dev/null 2>&1

Секретный ключ можно задать вне БД через константу в wp-config.php:

  define('SD_SIGNING_SECRET', '...');
  define('SD_ENDPOINT_URL', 'https://...');
