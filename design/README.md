# DataBridge — пакет вигрузки

Централізоване керування контактними даними мережі сайтів + WP-плагін-приймач.

## Що всередині
| Шлях | Опис |
|---|---|
| `CRM v2.dc.html` | **Панель керування** (робочий продукт, UA). Відкривається у браузері. |
| `DataBridge WP Plugin.dc.html` | **Плагін-приймач** для сайту (RU, стиль wp-admin). |
| `DataBridge Logo.dc.html` | Дослідження знака (фінал: «база → мережа»). |
| `CRM.dc.html` | Перша ітерація CRM (архів). |
| `docs/` | Специфікація (21 файл): огляд, глосарій, модель даних, бізнес-правила, екрани CRM і плагіна. |
| `CLAUDE.md` | Дизайн-бриф проєкту. |
| `support.js` | Рантайм Design Components (потрібен, щоб .dc.html відкривалися). |

## Як відкрити
Будь-який `*.dc.html` відкривається прямо у браузері (поряд має лежати `support.js`).

## Точка входу для SDD / специфікації
Читати в порядку:
1. `docs/README.md` — карта + інваріанти;
2. `docs/glossary.md` — терміни;
3. `docs/data-model.md` — сутності й поля;
4. `docs/business-rules.md` — правила показу/резолву/гео/доступу;
5. `docs/CRM/*`, `docs/Plugin/*` — екрани;
6. `.dc.html` — референс-реалізація (приклад, не джерело істини).

## Статус
Готово до вигрузки. Відкладено (за запитом): реальні форми додавання записів для цін/соцмереж/адрес — зараз тост-заглушка; месенджер уже має повну форму.
