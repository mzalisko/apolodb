# Ціни — слот із гео-варіантами
Ключ головний (`price_callout`…). Ціна ≠ одне число: слот містить варіанти (UA → 450 грн, PL → 120 zł, ALL → 500 грн).

## Таблиця
Значення = чипи «країна · ціна» (ALL нейтральний, ONLY жовтий, EXCEPT червоний), гео-пілюлі, статус.

## Бокова панель
- Список варіантів: клік — редагується (позначка), × видалити (мін. 1), «+ Гео-варіант».
- Поля варіанта: **число (обов'язкове**, помилка при збереженні без числа) + **валюта — довільний текст, опційно** (порожньо = лише число).
- Гео-режим варіанта + країни; живе прев'ю.

## Резолв
Перший варіант зі збігом гео для країни → його ціна; інакше ALL; якщо немає — ціни немає.
Мітка показу всім — завжди **ALL**.
