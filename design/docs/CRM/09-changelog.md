# Журнал змін
Три рівні, що комбінуються: **Усі сайти** / **По сайту** / **По типу даних** (чипи з лічильниками).

## Контекст сайту
У режимі «По сайту» вгорі — видимий банер «Журнал сайту: <назва> · домен» + «← Усі сайти». З огляду сайту «увесь журнал →» відкриває одразу цей рівень (піддомен → батьківський сайт).

## Подача
Групування по днях («Сьогодні», «Вчора» …) з лічильником. Запис: час+хто · сайт (клікабельний → його журнал) · тип-бейдж+поле · було→стало (сіра перекреслена → зелена; підсвічена змінена частина; негативні дії червоні; нові — бейдж «додано»). Масова дія — один запис. Порожній стан.
