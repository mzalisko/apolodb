# Масові зміни (майстер, 4 кроки)
Розрахований на 500+ сайтів.

## Крок 1 — Область
Пошук по сайтах/піддоменах · фільтр групи · «+ Усі показані», «★ Обране», «Усі активні», «Очистити» · перемикач «сайт разом із піддоменами». Черга обраних — чипами (× зняти).

## Крок 2 — Дані
Тип (телефони/месенджери/ціни/соцмережі) → поле: значення · Гео-варіант ціни (для цін) · гео-правило.

## Крок 3 — Зміна
- Значення: **однакове** / **індивідуально по сайтах** / **знайти → замінити**.
- Ціни: тільки встановлення значення (без ±%/±сума) + область варіантів: усі / лише ALL / варіант обраної країни.
- «Гео-варіант ціни»: число + валюта + країни — створює/оновлює варіант на кожному сайті.
- Гео-правило: 3 режими + модалка країн.

## Крок 4 — Перевірка
Таблиця «сайт · елементів · було → стане», підсумок, «Застосувати до N сайтів» → один запис у журнал + тост.
