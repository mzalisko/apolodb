# Телефони (еталонний тип)
Первинний рядок слота — **ключ** (моно, напр. `call_ua_01`); назва другорядна; декоративних підписів немає.

## Таблиця
Чекбокс · ключ+назва · активний номер (маска + канонічний) з бейджами месенджерів · ланцюг резервів (компактні позначки A/P1/P2) · гео (іконка + пілюлі країн, ALL/дозволено-жовті/заборонено-червоні-закреслені) · приховати(око) · шеврон.

## Бокова панель (єдиний патерн)
- **ID слота** (редагований) + підказка `[sd_phone id="…"]`; назва (необов'язкова).
- Канонічний E.164; маска з живим прев'ю.
- Гео-режим (3 картки) + «Керувати» → модалка країн.
- **Ланцюг резервів**: кожен рядок — редаговане поле номера, ↑/↓ (ролі перенумеровуються), × видалити; «+ Резерв». Резерв підміняє активний за недоступності.
- Прикріплення месенджерів: клік по наявних (мультивибір).
- Прив'язані адреси (один номер → кілька).
- Стани: «Є незбережені зміни» / «Збережено»; Esc закриває.

## Додавання
«+ Слот телефону» → бокова панель нового слота (порожній редагований ключ + дефолти).

## Масові дії (виділені рядки)
Змінити значення · змінити гео (повний вибір режиму+країн) · видалити — з прев'ю «N елементів».
