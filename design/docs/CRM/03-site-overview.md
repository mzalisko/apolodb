# Огляд сайту
Постійна смуга вкладок: Огляд · Телефони · Месенджери · Ціни · Соцмережі · Адреси (лічильники) — перемикання типів в один клік.

## «Контакти для «<Країна>»» — головне вікно
Селектор країни вгорі. Повноширинна таблиця з вирівняними колонками:
**Ключ · Значення · Гео-залежність · Статус**.
- Ключ (моno) головний; людська назва й ключ — у тултіпі при наведенні.
- Значення: номер у масці (той, що реально віддається — активний або резерв), біля нього бейджі прикріплених месенджерів (TG/VB/WA). Ціни — чипами гео-варіантів (ALL / країна · ціна), активний для країни підсвічений.
- Гео-залежність: єдині теги **ALL** / **ТІЛЬКИ …** / **КРІМ …**.
- Статус: показано / вимкнено / приховано (кольорова смуга статусу зліва рядка).

## Останні зміни на сайті
Стрічка «поле · було → стало» з дією **↩ Повернути** на кожному записі (тост). «Увесь журнал →» відкриває журнал, відфільтрований по цьому сайту (для піддомену — по батьківському).

## Токен і синхронізація
Токен (показати/сховати/копіювати, відкликати/оновити) і час останньої синхронізації. Дублів немає.
