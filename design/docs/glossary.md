# Глосарій (єдина термінологія)

- **Сайт** — домен у мережі. Може мати **піддомени**, завжди закріплені за батьківським сайтом.
- **Група** — довільне об'єднання сайтів; сайт може бути в кількох групах.
- **Слот** — одиниця даних певного типу на сайті (телефон, месенджер, ціна, соцмережа, адреса).
- **Ключ (key / id)** — незмінний slug слота, напр. `call_ua_01`, `price_diag`, `tg_dept_ua`. Головний ідентифікатор у списках, огляді й у виводі через шорткод/PHP. Людська назва — другорядна.
- **Канонічний номер** — телефон у форматі E.164 (`+380442238877`), без пробілів і символів.
- **Маска відображення** — шаблон показу, напр. `+38 (0##) ###-##-##`; бере останні N цифр канонічного номера.
- **Гео-режим / гео-залежність** — правило показу елемента. Єдині мітки:
  - **ALL** — показувати всім;
  - **ONLY** (у UI тег «ТІЛЬКИ …») — тільки переліченим країнам;
  - **EXCEPT** (тег «КРІМ …») — усім, крім перелічених країн.
- **Ланцюг резервів** — впорядкований список номерів слота: активний → резерв 1 → резерв 2. Сайт віддає перший доступний; резерв **підміняє** активний, а не є окремим слотом.
- **Гео-варіант ціни** — один зі значень ціни-слота для певного гео (UA → 450, PL → 120, ALL → 500).
- **Синхронізація** — отримання оновлень даних плагіном за токеном.
- **Токен** — ключ доступу сайту до панелі.
