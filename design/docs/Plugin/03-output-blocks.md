# Вкладка «Блоки вывода» (головна цінність)
Готові блоки: PHP у шаблони або шорткоди в контент/ACF. Перемикач «Шорткод | PHP».
Ключі слотів збігаються з CRM (`call_ua_01`, `price_callout`, `tg_dept_ua`, `fb_main`, `addr_kyiv`…).

## Стилізація (клас-префікс + власний клас)
- **Префікс CSS-класу** (глобально, напр. `db`) — кожен блок отримує базовий клас `db-phone`, `db-price`, `db-messenger`, `db-social`, `db-address`.
- **Власний клас** на кожен блок — додатковий клас для стилізації в темі.
- Клас потрапляє у вивід: шорткод `class="db-phone my-class"`, PHP `[ 'class' => 'db-phone my-class' ]`.

## 9 наборів
1. Номер — текст с маской · `[sd_phone id="…"]`
2. Номер — ссылка tel: · `output="link"`
3. Номер — сырой E.164 · `format="raw"`
4. Номер + мессенджеры · `messengers="yes"`
5. Цена — число · `[sd_price id="…"]`
6. Цена с валютой · `format="full"`
7. Мессенджер — ссылка · `[sd_messenger id="…"]`
8. Соцсеть — ссылка · `[sd_social id="…"]`
9. Адрес — текст · `[sd_address id="…"]`

Кожен блок: селект слота (за ключем) → код; поле «CSS-клас»; кнопка копіювання (✓ + тост); рядок «Выведет:» — живий результат з урахуванням гео-симулятора.
