# Вкладка «Данные» + гео-симулятор
Гео-симулятор: перемикач + «Показ как посетитель из: <страна>» — впливає на колонку «Сейчас выводится» і прев'ю блоків.
Типи чипами (Телефоны/Цены/Мессенджеры/Соцсети/Адреса). Колонки: слот (назва+ключ, збігається з CRM) · значення · гео · «Сейчас выводится».
Телефон: «основной» (зелений) / «резерв N» (бурштин, коли основной недоступен) / «скрыт». Ціни — чипи гео-варіантів, активний зелений. Адреси — прив'язаний номер (☎) + графік. Легенда під таблицею.

## Иконки мессенджеров
Для месенджерів — панель вибору іконки на канал (TG/VB/WA): **«Буквы»** (двобуквений бейдж із кольором каналу) або **«Картинка»** (власна завантажена іконка). Вибір враховується у виводі.
