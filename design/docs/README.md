## Ключові інваріанти (швидко)
- Телефон зберігається канонічно (E.164); показ — через маску-шаблон.
- Кожен елемент має **ключ** (slug, напр. `call_ua_01`) — головний ідентифікатор і в UI, і у виводі.
- Гео-режим елемента: **ALL** (усім) / **ONLY** (тільки переліченим) / **EXCEPT** (усім, крім перелічених).
- Слот телефону = ланцюг: активний → резерв 1 → резерв 2; сайт віддає перший доступний.
- Ціна = слот із гео-варіантами (різна ціна різним країнам) + варіант ALL за замовчуванням.
